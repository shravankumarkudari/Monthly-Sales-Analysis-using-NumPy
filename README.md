# 📊 Monthly Sales Analysis using NumPy

## 📌 Project Overview

This is a simple Python mini project that analyzes monthly sales data using the powerful **NumPy** library.
The program takes user input for sales across all 12 months and provides useful insights like total sales, average, highest and lowest months, and comparisons with average performance.

---

## 🔢 What is NumPy?

NumPy (Numerical Python) is a popular Python library used for:

* Fast mathematical computations
* Working with arrays and matrices
* Performing statistical operations

### 🔑 Key Features:

* Efficient array operations
* Built-in functions like `sum()`, `mean()`, `max()`, `min()`
* Faster than traditional Python lists for numerical data

---

## 💡 Project Features

This mini project performs the following analysis:

* ✅ Takes monthly sales input from the user
* ✅ Calculates:

  * Total yearly sales
  * Average monthly sales
  * Highest sales value
  * Lowest sales value
* ✅ Identifies:

  * Best performing month 📈
  * Worst performing month 📉
* ✅ Shows:

  * Months with above-average sales
  * Months with below-average sales
